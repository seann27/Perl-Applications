The script format_converter.pl takes two command line arguments.

1) The file the user wants to convert
2) The format the user wants to convert to

The examples provided were generated by using a fasta file as the original file and converting to pir and genbank formats.

$ perl format_converter.pl ./RPE65.fasta pir

output: RPE65.pir

$ perl format_converter.pl ./RPE65.fasta genbank

output: RPE65.genbank
